# 生理周期トラッカー【無料】次回予測・体調管理 - 技術仕様書

## 概要

**サービス名**: Menstrual Cycle Tracker
**バージョン**: 1.0.0
**更新日**: 2026-05-27
**URL**: https://appadaycreator.com/menstrual-cycle-tracker/

生理周期を記録して次回予測日・排卵日を計算。体調の波を把握して毎月の計画を立てやすくするシンプルな周期管理ツール。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 「最終生理開始日」に最後に生理が始まった日を入力する
2. スライダーで平均生理周期（日数）を設定する（標準28日）
3. 生理期間（日数）を設定する（標準5日）
4. 「計算する」ボタンをクリックする
5. 今後3ヶ月の生理予定日・排卵予定日・妊娠可能期間を確認する

## よくある質問（FAQ）

**Q: 生理周期トラッカーは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 入力した日付データはどこかに送信されますか？**

いいえ。すべての計算はブラウザ内で完結し、サーバーへのデータ送信・保存は行いません。

**Q: 排卵日の計算精度はどの程度ですか？**

平均周期をもとにした参考値です。実際の排卵日は体調・ストレスで変動します。妊活・避妊目的での使用には、必ず医師または基礎体温測定との併用をご検討ください。

**Q: 周期が不規則な場合はどうすればいいですか？**

直近3ヶ月の平均を計算して入力してみてください。28日が標準ですが、21〜40日程度の幅があります。

**Q: 生理が来ない場合や周期が大きく変わった場合は？**

ストレス・体重変化・病気・妊娠などが原因の可能性があります。3ヶ月以上異常が続く場合は婦人科への相談をおすすめします。
