## v2.0.0 改善内容（2026-05-29）

### P1: localStorage永続化
- 入力値（最終生理日・周期・生理期間・過去3件の生理日）をlocalStorageへ自動保存（キー: `mct_v2`）
- ページ読み込み時に自動復元し、最終生理日が保存済みの場合は自動で計算結果を表示
- リセットボタンでlocalStorageを完全削除

### P2: 今日の周期状態カード
- 現在の周期フェーズ（月経期/卵胞期/排卵期/黄体期）をカラーバッジで表示
- 今日が周期何日目かを表示
- 次の生理まであと何日・次の排卵まであと何日をカウントダウン形式で表示
- フェーズ判定: 月経期=day1〜periodDays、排卵期=ovulDay±3、黄体期=それ以降

### P3: 2ヶ月ミニカレンダー
- 現在月と翌月のカレンダーを2列表示（モバイルは縦積み）
- 生理予定日（赤）・妊娠可能期間（ピンク）・排卵予定日（紫）を色分け
- 今日は赤枠で強調
- 凡例付き

### P4: 過去の生理日から平均周期を自動計算
- アコーディオン形式で3件の過去生理開始日入力欄を追加
- 2件以上入力すると各間隔の平均を自動計算してスライダーに反映
- 21〜40日の範囲外は警告表示

### P5: スタイル済みスライダー + 結果コピーボタン
- レンジスライダーをカスタムCSSでスタイリング（赤グラデーショントラック・白ツマミ）
- 計算結果に「📋 結果をコピー」ボタンを追加（Clipboard API + fallback対応）
- コピー成功時にボタンが「✅ コピーしました」に一時変化

### 予測ロジック改善
- 「Nヶ月後」表示を「次の生理/2回後/3回後」に変更
- 現在日付を基準に次回以降の実際の生理サイクルを表示（過去分をスキップ）
**URL**: https://appadaycreator.com/menstrual-cycle-tracker/

生理周期を記録して次回予測日・排卵日を計算。体調の波を把握して毎月の計画を立てやすくするシンプルな周期管理ツール。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 「最終生理開始日」に最後に生理が始まった日を入力する
2. スライダーで平均生理周期（日数）を設定する（標準28日）
3. 生理期間（日数）を設定する（標準5日）
4. 「計算する」ボタンをクリックする
5. 今後3ヶ月の生理予定日・排卵予定日・妊娠可能期間を確認する

## よくある質問（FAQ）

**Q: 生理周期トラッカーは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 入力した日付データはどこかに送信されますか？**

いいえ。すべての計算はブラウザ内で完結し、サーバーへのデータ送信・保存は行いません。

**Q: 排卵日の計算精度はどの程度ですか？**

平均周期をもとにした参考値です。実際の排卵日は体調・ストレスで変動します。妊活・避妊目的での使用には、必ず医師または基礎体温測定との併用をご検討ください。

**Q: 周期が不規則な場合はどうすればいいですか？**

直近3ヶ月の平均を計算して入力してみてください。28日が標準ですが、21〜40日程度の幅があります。

**Q: 生理が来ない場合や周期が大きく変わった場合は？**

ストレス・体重変化・病気・妊娠などが原因の可能性があります。3ヶ月以上異常が続く場合は婦人科への相談をおすすめします。
